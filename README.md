# وازن Wazen

نظام ويب عربي لإدارة الراتب والمصروفات والأهداف المالية حتى الراتب القادم.

## فكرة النظام

وازن ليس تطبيق محاسبة معقد. هو مركز تحكم شخصي يركز على:

- إدخال الراتب والدخل الإضافي.
- تسجيل مصروف سريع جدًا من الصفحة الرئيسية.
- إضافة أهداف مالية متعددة.
- تقسيم الالتزامات إلى مستمرة ومؤقتة.
- حساب المتاح الحقيقي للصرف.
- حساب المبلغ المناسب يوميًا حتى الراتب القادم.
- مساعد ذكي محلي يحلل بناءً على البيانات المدخلة فقط.

## التشغيل على GitHub Pages

ارفع الملفات التالية إلى المستودع مباشرة:

- `index.html`
- `style.css`
- `app.js`
- `README.md`

ثم من GitHub:

Settings → Pages → Deploy from a branch → main → /root → Save
